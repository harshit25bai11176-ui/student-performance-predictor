# student-performance-predictor
## 📌 Overview
This project predicts a student's performance based on study hours and attendance using Machine Learning.

## ❓ Problem Statement
Students often cannot estimate their exam performance based on their preparation.

## 💡 Solution
A Machine Learning model is used to predict marks and determine whether a student will pass or fail.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Features
- Predicts student marks
- Shows Pass/Fail result
- Simple and easy to use

## ▶️ How to Run
1. Install dependencies:
   pip install pandas numpy scikit-learn matplotlib

2. Run the program:
   python main.py

## 📁 Project Structure
- main.py → Main code
- data.csv → Dataset
